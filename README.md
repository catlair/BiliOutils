## 开发

安装依赖

```bash
yarn install
```

开发环境运行

```bash
yarn dev
```

检查和格式化代码

```bash
yarn lint:prettier
yarn lint:eslint
```

开发环境模仿 npm 全局命令

```bash
yarn dev:bin -c ./config/config.json
```

编译

```bash
yarn build
```
